This domain, called "spanner," involves a man named Bob who can walk between locations and carry spanners. There are also nuts that need to be tightened using a spanner. The domain includes predicates such as "at" for locating objects, "carrying" for the man's possession of a spanner, "useable" for whether a spanner can be used, "link" for connecting locations, "tightened" for a nut that has been tightened, and "loose" for a nut that needs to be tightened. The actions available in this domain include walking, picking up a spanner, and tightening a nut using a spanner. The given problems involve placing Bob, nuts, and spanners in different locations and requiring that all the nuts be tightened.
